URL MODEL V2 FOR MIRAGE

OVERVIEW
--------
This model detects phishing URLs using lexical URL features only.

The model does NOT use any PhiUSIIL dataset-specific features.
All features can be generated directly from a URL string during runtime.

MODEL
-----
Algorithm:
XGBoost Classifier

Dataset:
PhiUSIIL_Phishing_URL_Dataset.csv

Samples:
235,795

Features:
19 lexical URL features

Accuracy:
99.56%

FEATURE ORDER
-------------
1. url_length
2. domain_length
3. tld_length
4. path_length
5. query_length
6. subdomain_count
7. slash_count
8. dot_count
9. hyphen_count
10. underscore_count
11. digit_count
12. letter_count
13. special_char_count
14. has_https
15. has_at_symbol
16. has_double_slash
17. is_ip_address
18. percent_encoding_count
19. entropy
